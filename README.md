### RBAC权限模型练习项目

#### 技术栈
**后端**
1. Spring
2. SpringMvc
3. Mybatis
4. JSP
5. java邮箱开发
6. RBAC权限模型

**前端**
1. UI框架：bootstrap
2. Ztree:显示权限树
4. Jquery ,Ajax
5. 弹出插件：Layer
#### 开发工具
1. 服务器：tomcat
2. 数据库：mysql
3. 集成坏境：IDEA
4. 可视化工具: SqlYog

#### 数据库设计
1. 权限表
2. 角色表
3. 用户表
（一个用户可以赋予多个角色，一个角色可以有多个权限）

*简单的一个权限练习的项目*。
